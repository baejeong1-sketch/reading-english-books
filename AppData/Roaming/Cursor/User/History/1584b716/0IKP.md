## JJ Reading English Books (제이 제이 독서 영어 웹)

어린이(유치 4세 ~ 초6)를 위한 영어 독서 웹 서비스의 프론트엔드 프로토타입입니다.

### 주요 기능 개요

- **회원가입/로그인 화면**
- **레벨 테스트 (1~10단계 권장 레벨 산출 예시)**
- **레벨/분야별 도서관 UI (예시 도서 데이터)**
- **도서 리더 화면**
  - 본문 문장 단위 표시
  - 단어 클릭 시 품사/뜻/발음 기호 표시 및 발음 듣기 버튼(데모 알림)
  - 독해 문제 출제 및 채점 + 틀린 문항 힌트 표시
  - 문장 패턴 빈칸 채우기 문제 및 채점
  - 쓰기 문제 출제, 교정 예시 및 따라 쓰기 영역
  - 녹음 UI 및 발음 피드백 예시(실제 녹음/인식 기능은 미구현)
- **마이페이지**
  - 레벨, 읽은 권수, 최근 도서 성취도 예시
  - 월별 테스트 및 레벨 변화 텍스트 요약
- **관리자 페이지**
  - 사용자 목록/요약
  - 월별 다독왕 TOP 3 & 포인트 예시
  - 월별 테스트 성적 요약

### 실행 방법

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 로 접속하면 화면을 확인할 수 있습니다.

### 앞으로 확장 시 고려사항

- 실제 10,000권 도서 데이터 및 레벨/연령별 메타데이터 설계
- 사용자/관리자 인증 및 권한(백엔드/DB 연동)
- 단어 사전/발음(TTS), 음성 녹음 및 발음 분석 API 연동
- 월별 테스트 문제 자동 생성 로직 및 성취도 통계/그래프
- 반응형/접근성 강화, 어린이용 일러스트/아이콘 추가
