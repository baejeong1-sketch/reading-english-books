# J J 독서 영어 - 10만 권 AI 통합 영어 독서 프로그램

유치부 4세부터 초등 6학년까지를 위한 종합 영어 독서 학습 플랫폼

## 주요 기능

### 1. 레벨 테스트 시스템
- 10단계 레벨 시스템
- 사용자 실력에 맞는 레벨 자동 배정

### 2. 도서 시스템
- 각 레벨별 10,000권의 도서 제공
- 레벨에 맞는 도서 자동 추천
- 단어 클릭 시 즉시 뜻 표시

### 3. 발음 교정 시스템
- 소리 내어 읽기 녹음
- AI 기반 발음 분석
- 잘못된 발음 단어 리스트 제공
- 발음 연습 및 재평가

### 4. 독해 문제 시스템
- 레벨별 난이도 조절 (1-10단계)
- 연령대별 맞춤 문제 생성
- 자동 채점 및 오답 분석
- 맞춤형 학습 팁 제공
- 완벽한 이해까지 반복 학습

### 5. 문장 패턴 연습
- 도서 기반 문장 패턴 학습
- 빈칸 채우기 연습
- 레벨 및 연령 고려

### 6. 쓰기 연습
- 도서 그림 및 내용 활용
- AI 기반 첨삭 지도
- 문법 교정 및 글의 흐름 개선

### 7. 월별 평가 시스템
- 월별 학습 내용 기반 테스트
- 레벨 업 판단 및 권장사항 제공

### 8. 학습 기록 관리
- 개인별 상세 학습 기록
- 읽은 도서 목록
- 녹음 파일 저장
- 독해, 쓰기, 문장 패턴 연습 기록
- 월별 테스트 결과

## 월 사용료

**권장 가격: 월 19,900원**

가격 산정 근거:
- 국내 유사 서비스: 월 15,000원 ~ 30,000원
- 해외 서비스 (Raz-Kids 등): 월 $10-20 (약 13,000원-26,000원)
- 제공 기능의 가치 (10만 권 도서, AI 발음 교정, 맞춤형 학습 등) 고려
- 유치부-초등학생 대상 프리미엄 서비스 포지셔닝

## 기술 스택

- **Frontend**: React 18, React Router
- **State Management**: Zustand
- **Build Tool**: Vite
- **Audio**: Web Speech API, React Audio Player
- **AI Integration**: (향후 OpenAI API 연동 예정)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 프로젝트 구조

```
src/
├── components/       # 재사용 가능한 컴포넌트
├── pages/           # 페이지 컴포넌트
├── hooks/           # 커스텀 훅
├── store/           # 상태 관리

├── services/        # API 서비스
└── styles/          # 스타일 파일
```
