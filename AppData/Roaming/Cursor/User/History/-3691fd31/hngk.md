# J J 독서 영어 - 10만 권 AI 통합 영어 독서 프로그램

유치부 4세부터 초등 6학년까지를 위한 종합 영어 독서 학습 플랫폼

## 주요 기능

### 0. 사용자 관리 시스템
- **가입 승인 시스템**: 사용자 가입 시 관리자 승인 필요
- **사용자 상태 관리**: 대기중, 승인됨, 거부됨, 만료됨 상태 관리
- **만기일 관리**: 가입일 및 만기일 기록 및 관리
- **만기일 알림**: 만기일 일주일 전 자동 알림 서비스
- **관리자 페이지**: 사용자 승인/거부, 만기일 연장, 사용자 관리 기능

### 1. 레벨 테스트 시스템
- 10단계 레벨 시스템
- 사용자 실력에 맞는 레벨 자동 배정

### 2. 도서 시스템
- 각 레벨별 10,000권의 도서 제공
- 레벨에 맞는 도서 자동 추천
- 단어 클릭 시 즉시 뜻 표시

### 3. 발음 교정 시스템
- 소리 내어 읽기 녹음
- AI 기반 발음 분석
- 잘못된 발음 단어 리스트 제공
- 발음 연습 및 재평가

### 4. 독해 문제 시스템
- 레벨별 난이도 조절 (1-10단계)
- 연령대별 맞춤 문제 생성
- 자동 채점 및 오답 분석
- 맞춤형 학습 팁 제공
- 완벽한 이해까지 반복 학습

### 5. 문장 패턴 연습
- 도서 기반 문장 패턴 학습
- 빈칸 채우기 연습
- 레벨 및 연령 고려

### 6. 쓰기 연습
- 도서 그림 및 내용 활용
- AI 기반 첨삭 지도
- 문법 교정 및 글의 흐름 개선

### 7. 월별 평가 시스템
- 월별 학습 내용 기반 테스트
- 레벨 업 판단 및 권장사항 제공

### 8. 학습 기록 관리
- 개인별 상세 학습 기록
- 읽은 도서 목록
- 녹음 파일 저장
- 독해, 쓰기, 문장 패턴 연습 기록
- 월별 테스트 결과

## 설치 및 실행

### ⚠️ 중요: Live Server로는 실행할 수 없습니다!

이 프로젝트는 React + Vite로 만들어졌으므로 **Vite 개발 서버**를 사용해야 합니다.

### 방법 1: 배치 파일 사용 (가장 쉬운 방법)

1. **INSTALL.bat** 파일을 더블클릭하여 의존성 설치
2. **START.bat** 파일을 더블클릭하여 개발 서버 시작
3. 브라우저가 자동으로 열립니다 (http://localhost:3000)

### 방법 2: 터미널 사용

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. 브라우저에서 http://localhost:3000 접속
```

### 프로덕션 빌드

```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

### 문제 해결

- **Node.js가 설치되어 있지 않은 경우**: https://nodejs.org/ 에서 설치
- **포트 3000이 사용 중인 경우**: `vite.config.js`에서 포트 번호 변경
- **의존성 설치 오류**: `node_modules` 폴더 삭제 후 `npm install` 다시 실행

│   ├── UserManagementPage.jsx  # 사용자 관리 페이지

## 사용자 상태 및 승인 프로세스

1. **가입 신청**: 사용자가 홈페이지에서 정보 입력 후 가입 신청
2. **대기 상태**: 가입 신청 후 `pending` 상태로 대기
3. **관리자 승인**: 관리자가 사용자를 승인하면 `approved` 상태로 변경 및 만기일 설정
4. **서비스 이용**: 승인된 사용자만 학습 기능 이용 가능
5. **만기일 알림**: 만기일 일주일 전 자동 알림 표시
6. **만료 처리**: 만기일이 지나면 자동으로 `expired` 상태로 변경
