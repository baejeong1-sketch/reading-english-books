# J J 독서 영어 - 10만 권 AI 통합 영어 독서 프로그램

유치부 4세부터 초등 6학년까지를 위한 종합 영어 독서 학습 플랫폼

## 주요 기능

### 0. 사용자 관리 시스템
- **가입 승인 시스템**: 사용자 가입 시 관리자 승인 필요
- **사용자 상태 관리**: 대기중, 승인됨, 거부됨, 만료됨 상태 관리
- **만기일 관리**: 가입일 및 만기일 기록 및 관리
- **만기일 알림**: 만기일 일주일 전 자동 알림 서비스
- **관리자 페이지**: 사용자 승인/거부, 만기일 연장, 사용자 관리 기능

### 1. 레벨 테스트 시스템
- 10단계 레벨 시스템
- 사용자 실력에 맞는 레벨 자동 배정

### 2. 도서 시스템
- 각 레벨별 10,000권의 도서 제공
- 레벨에 맞는 도서 자동 추천
- 단어 클릭 시 즉시 뜻 표시

### 3. 발음 교정 시스템
- 소리 내어 읽기 녹음
- AI 기반 발음 분석
- 잘못된 발음 단어 리스트 제공
- 발음 연습 및 재평가

### 4. 독해 문제 시스템
- 레벨별 난이도 조절 (1-10단계)
- 연령대별 맞춤 문제 생성
- 자동 채점 및 오답 분석
- 맞춤형 학습 팁 제공
- 완벽한 이해까지 반복 학습

### 5. 문장 패턴 연습
- 도서 기반 문장 패턴 학습
- 빈칸 채우기 연습
- 레벨 및 연령 고려

### 6. 쓰기 연습
- 도서 그림 및 내용 활용
- AI 기반 첨삭 지도
- 문법 교정 및 글의 흐름 개선

### 7. 월별 평가 시스템
- 월별 학습 내용 기반 테스트
- 레벨 업 판단 및 권장사항 제공

### 8. 학습 기록 관리
- 개인별 상세 학습 기록
- 읽은 도서 목록
- 녹음 파일 저장
- 독해, 쓰기, 문장 패턴 연습 기록
- 월별 테스트 결과

## 관리자 기능

### 관리자 로그인
- 이메일: `admin@jjreading.com`
- 비밀번호: `admin123`
- 실제 운영 시에는 안전한 인증 시스템으로 교체 필요

### 관리자 페이지 기능
1. **사용자 승인/거부**
   - 가입 신청 대기 중인 사용자 승인
   - 거부 시 사유 입력 가능
   - 승인 시 기본 1개월 사용 기간 부여

2. **사용자 관리**
   - 전체 사용자 목록 조회
   - 가입일, 만기일 기록 및 관리
   - 만기일 연장 기능 (1~12개월)
   - 사용자 검색 및 정렬 기능
   - 만기일 임박 사용자 알림

3. **통계 정보**
   - 전체 사용자 수
   - 상태별 사용자 수 (대기중, 승인됨, 만료됨)
   - 만기일 임박 사용자 수

## 기술 스택

- **Frontend**: React 18, React Router
- **State Management**: Zustand
- **Build Tool**: Vite
- **Audio**: Web Speech API, React Audio Player
- **AI Integration**: (향후 OpenAI API 연동 예정)

## 프로젝트 구조

```
src/
├── components/              # 재사용 가능한 컴포넌트
│   ├── Layout.jsx          # 메인 레이아웃
│   ├── ProtectedRoute.jsx   # 승인된 사용자만 접근 가능한 라우트 보호
│   └── ExpiryNotification.jsx  # 만기일 알림 컴포넌트
├── pages/                   # 페이지 컴포넌트
│   ├── HomePage.jsx         # 홈페이지 (가입)
│   ├── PendingApprovalPage.jsx  # 승인 대기 페이지
│   ├── AdminPage.jsx        # 관리자 페이지 (사용자 승인)
│   ├── UserManagementPage.jsx  # 사용자 관리 페이지
│   └── ... (기타 학습 페이지들)
├── store/                   # 상태 관리 (Zustand)
│   ├── userStore.js         # 현재 사용자 정보
│   ├── adminStore.js       # 관리자 정보
│   ├── usersStore.js       # 전체 사용자 관리
│   └── learningStore.js    # 학습 기록
├── utils/                   # 유틸리티 함수
└── styles/                  # 스타일 파일
```
