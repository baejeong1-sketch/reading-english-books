# VS Code에서 실행하는 방법

## 🎯 가장 쉬운 방법

### 방법 1: F5 키 사용
1. VS Code에서 프로젝트 폴더 열기
2. **F5 키**를 누르기
3. "Vite 개발 서버 시작" 선택
4. 브라우저가 자동으로 열립니다!

### 방법 2: 실행 및 디버그 메뉴
1. 왼쪽 사이드바에서 **실행 및 디버그** 아이콘 클릭 (또는 `Ctrl+Shift+D`)
2. 상단에서 "Vite 개발 서버 시작" 선택
3. ▶️ 버튼 클릭

### 방법 3: 터미널 사용
1. `Ctrl + ~` (백틱)로 터미널 열기
2. 다음 명령어 입력:
   ```bash
   npm run dev
   ```

### 방법 4: 작업 실행
1. `Ctrl+Shift+P`로 명령 팔레트 열기
2. "Tasks: Run Task" 입력
3. "npm: dev" 선택

## ⚠️ Live Server 사용 불가

**Live Server 확장 프로그램으로는 실행할 수 없습니다!**

이유:
- React 앱은 빌드 과정이 필요합니다
- JSX 파일은 브라우저가 직접 실행할 수 없습니다
- Vite가 JSX를 JavaScript로 변환해야 합니다

## ✅ 올바른 방법

**Vite 개발 서버**를 사용해야 합니다:
- 핫 리로드 지원
- 빠른 빌드
- 개발 도구 지원

## 📝 팁

- 개발 서버는 파일을 저장하면 자동으로 새로고침됩니다
- 서버를 중지하려면 터미널에서 `Ctrl + C`를 누르세요
