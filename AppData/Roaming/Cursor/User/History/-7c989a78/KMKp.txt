==========================================
J J 독서 영어 - 실행 방법
==========================================

⚠️ 주의: 이 프로젝트는 React + Vite로 만들어졌습니다.
Live Server로는 실행할 수 없습니다!

==========================================
1단계: 의존성 설치
==========================================

터미널(명령 프롬프트 또는 PowerShell)을 열고
프로젝트 폴더로 이동한 후 다음 명령어를 실행하세요:

npm install

==========================================
2단계: 개발 서버 실행
==========================================

의존성 설치가 완료되면 다음 명령어를 실행하세요:

npm run dev

==========================================
3단계: 브라우저에서 확인
==========================================

개발 서버가 실행되면 자동으로 브라우저가 열립니다.
만약 자동으로 열리지 않으면 다음 주소로 접속하세요:

http://localhost:3000

==========================================
문제 해결
==========================================

1. npm이 설치되어 있지 않은 경우:
   - Node.js를 설치하세요: https://nodejs.org/
   - 설치 후 터미널을 다시 시작하세요

2. 포트 3000이 이미 사용 중인 경우:
   - vite.config.js에서 포트 번호를 변경할 수 있습니다

3. 에러가 발생하는 경우:
   - node_modules 폴더를 삭제하고
   - npm install을 다시 실행하세요
